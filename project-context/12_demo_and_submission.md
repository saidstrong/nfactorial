## README project description

BLACKOUT RAID is an AI-directed cyber roguelite arena shooter. The player enters a corrupted digital dungeon, clears enemy waves, chooses upgrades, and fights the Blackout Core, a hard adaptive boss. OpenAI acts as a Raid Director by generating mission briefings, crisis events, boss phase decisions, and final debriefs. The game is built with Next.js, TypeScript, Tailwind, Phaser, and OpenAI API, with optional Supabase support for future room-code co-op.
