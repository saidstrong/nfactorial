## README short description
BLACKOUT GRID is an AI-powered crisis-control game based on Minesweeper. The player restores a corrupted city power grid by revealing safe nodes, quarantining threats, and using limited AI tools. The game combines deterministic gameplay with OpenAI-generated briefings, tactical counsel, and mission debriefs, plus a Supabase-powered leaderboard.
